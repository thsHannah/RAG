# 牛肉麵
牛肉麵是台灣最具代表性的麵食之一，起源於1950年代隨國民政府來台的四川移民。台北市每年舉辦牛肉麵節，吸引大批饕客。正宗台灣牛肉麵分為紅燒與清燉兩大派系：紅燒牛肉麵湯頭以豆瓣醬、辣椒、番茄為基底，色澤深紅、香氣濃郁；清燉牛肉麵湯色清澈，以牛骨熬煮數小時，鮮甜回甘。常見配料包括牛腱、牛肋條、牛腩，搭配手工拉麵或刀削麵。知名店家有林東芳牛肉麵、永康牛肉麵。一碗正宗牛肉麵的價格約150至350元不等。

# 小籠包
小籠包源自江南，但在台灣發揚光大，以台北鼎泰豐為代表揚名國際。台灣小籠包的特色在於薄皮多汁，每顆標準重量為21克，需要18褶以上。內餡以豬絞肉加入豬皮凍製成，加熱後皮凍融化形成湯汁。正確吃法：以筷子夾起後輕放湯匙，咬一小口讓湯汁流入湯匙，再加薑絲與鎮江醋。鼎泰豐在米其林指南中屢獲星級評鑑，分店遍布全球。台北、台中、高雄皆有知名小籠包店。

# 臭豆腐
臭豆腐是台灣夜市不可缺少的小吃，以發酵過的豆腐油炸或燒烤製成。台式炸臭豆腐外酥內軟，搭配泡菜與蒜泥醬油食用，是台灣夜市的靈魂食物之一。臭豆腐的特殊氣味來自發酵過程中產生的硫化物，雖然氣味強烈，但愛好者眾多。麻辣臭豆腐是另一種受歡迎的形式，以麻辣鍋底燉煮，口感更為豐富。台北寧夏夜市、士林夜市、基隆廟口都是品嚐臭豆腐的好去處。

# 珍珠奶茶
珍珠奶茶（bubble tea）發源於1980年代台灣台中，由春水堂與翰林茶館各自宣稱為發明者。珍珠是以樹薯粉（木薯澱粉）製成的黑色圓球，煮熟後加入砂糖與蜂蜜增添甜味，口感Q彈有嚼勁。傳統珍珠奶茶以紅茶加鮮奶或奶精調製，現代變化包括抹茶珍珠、芋頭珍珠、黑糖虎紋奶茶等。全台連鎖品牌包括50嵐、CoCo都可、茶湯會、貢茶等。台灣珍珠奶茶文化已影響全球，在歐美亞各地掀起流行風潮。

# 鹽酥雞
鹽酥雞是台灣最普遍的炸物小吃，起源於1970年代台灣，是夜市與路邊攤的固定班底。做法是將雞肉、雞排或各式食材醃製後裹粉酥炸，最後加入九層塔大火快炸，撒上胡椒鹽與辣椒粉。除了雞肉，常見的鹽酥品項還有甜不辣、雞腸、玉米、杏鮑菇、皮蛋等。台灣各地都有知名的鹽酥雞攤，尤以夜市攤車最為普遍，點一份約50至100元。在台灣，鹽酥雞幾乎24小時都買得到，是最受歡迎的宵夜選擇之一。

# 肉圓
肉圓（ba-wan）是台灣中部發源的傳統小吃，外皮由番薯粉製成，呈半透明狀，內餡包含豬絞肉、筍丁、香菇。烹調方式分為油炸與清蒸兩種，彰化肉圓以油炸為主，北部肉圓則多為清蒸。吃時淋上以甜辣醬、米醬調製的醬汁，口感軟Q香甜。彰化是肉圓的發源地，當地有許多百年老店，如阿璋肉圓。台灣各縣市的肉圓口味與製法略有不同，形成各具特色的地方風味。

# 刈包
刈包（gua-bao）俗稱「虎咬豬」，是台灣傳統街頭小吃，近年來也成為國際知名的台灣美食符號。由半月形鬆軟的白麵包夾入滷豬五花肉、酸菜、香菜、花生粉而成。軟嫩的麵包搭配香Q的豬肉與酸甜配料，層次豐富。刈包在台灣尾牙與農曆年節文化中扮演重要角色，象徵「發財」。近年在紐約、倫敦等國際城市，刈包以「Taiwanese hamburger」之姿走紅，成為台灣飲食外交的代表之一。

# 蚵仔煎
蚵仔煎是台灣與閩南地區的傳統小吃，在台灣夜市極為普遍。以新鮮牡蠣（蚵仔）、雞蛋、地瓜粉漿混合煎製，再淋上甜辣醬與海山醬，加入高麗菜或茼蒿增色。正宗蚵仔煎的蚵仔必須新鮮，外皮微酥內裡軟嫩，煎製過程中加水使粉漿形成特殊的黏稠口感。台南、嘉義、台北士林夜市的蚵仔煎各有特色。台南蚵仔煎偏甜，使用較多醬料；北部則較鹹香。

# 台南擔仔麵
擔仔麵是台南最具代表性的傳統小吃，由洪姓漁民於清朝光緒年間創立，因漁民在淡季挑擔叫賣麵食而得名。正宗擔仔麵以細油麵為底，澆上以豬骨與蝦頭熬製的高湯，放上滷肉燥、鮮蝦、豆芽菜、香菜，再淋一匙自製蝦膏。份量小巧，通常一碗僅50至60元，適合搭配其他台南小吃一同品嚐。度小月擔仔麵是百年老店，是台南飲食文化的活歷史。

# 芒果冰
台灣芒果冰是享譽國際的夏季甜品，以台灣特產愛文芒果製作，果肉香甜多汁、纖維細膩。台式芒果冰分為刨冰版與雪花冰版：刨冰以粗冰刨製，口感較粗獷；雪花冰由牛奶凍製再刨成薄片，口感如棉花般細緻。標準配料包括新鮮芒果切丁、芒果冰淇淋、芒果醬，部分店家還加入西米露或珍珠。台南與台中是芒果冰的重鎮，每年五至九月是芒果季，台南正興街的冰品店常大排長龍。一碗約100至200元。

# 台灣滷肉飯
滷肉飯（魯肉飯）是台灣最具代表性的國民美食，幾乎每間台式快餐店、自助餐都有供應。以豬五花肉切小丁後加入醬油、米酒、糖、五香粉、紅蔥頭長時間滷製，湯汁濃稠油亮，淋在白米飯上，搭配滷蛋、醃黃蘿蔔。台北與台南的滷肉飯有所不同：台北版本肉丁較細、醬汁偏甜；台南人習慣稱之為「肉燥飯」，肉末更細、風味偏鹹香。一碗滷肉飯約35至60元，是台灣最親民的美食之一。

# 葱抓餅
葱抓餅是台灣街頭早餐與夜市的人氣美食，以麵團加入豬油與葱花層層折疊，用鐵板煎至外酥內軟，可加蛋、加起司、加肉鬆等各種配料。製作時師傅用手「抓」鬆麵皮，讓餅皮產生多層次的酥脆感，因此得名。葱抓餅通常搭配甜辣醬或沙茶醬食用。在台灣各大學附近、捷運站出口常見葱抓餅攤車，是學生族與上班族最愛的快速早餐，一份約40至70元。

# 鳳梨酥
鳳梨酥是台灣最知名的伴手禮點心，外皮由奶油、麵粉、奶粉製成，酥鬆入口即化；內餡傳統上以冬瓜與鳳梨混合製成，近年流行「土鳳梨酥」，以100%純鳳梨製餡，酸甜滋味更為自然。鳳梨在台語中諧音「旺來」，象徵好運與繁榮，因此成為重要節慶與商務往來的贈禮首選。台北的微熱山丘、犁記、台中的一之軒、宝泉等都是知名品牌。每盒10入的鳳梨酥約300至600元不等。

# 三杯雞
三杯雞是台灣家常料理的代表菜色之一，以一杯麻油、一杯米酒、一杯醬油為基礎調味（現代版本比例有所調整），與雞肉、老薑、大蒜、辣椒一同爆炒，起鍋前大量加入九層塔，香氣四溢。三杯雞的精髓在於大火快炒收汁，讓醬汁均勻裹附雞肉，色澤亮麗、鹹甜適中、帶有九層塔的獨特香氣。此料理源於江西，在台灣經過改良成為廣受喜愛的家常菜，在台式餐廳菜單上幾乎都能見到。

# 鐵板燒
台式鐵板燒與日式不同，屬於平價快餐形式，是台灣校園周邊與夜市的常見餐食。以鐵板盛裝炒麵或炒飯，搭配豬排、牛排、雞排或海鮮，旁邊附湯、鐵板麵、玉米、奶油等配菜。台式鐵板燒價格親民，一份套餐約120至200元，是高CP值的飽足感餐食。特色在於主廚在顧客面前現場烹調，鐵板滋滋作響，視覺與嗅覺享受兼具，是台灣獨特的飲食文化。
